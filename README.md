# 🤖 Arabic Question Answering with Transformers

## Overview
This project demonstrates Arabic question answering using transformer-based NLP models. Given an Arabic context and a question, the model predicts the most relevant answer span automatically.

## Problem
Question answering is an important NLP task that helps users retrieve precise information from text. Arabic QA is particularly valuable because it supports language understanding for Arabic content and applications.

## Approach
- Prepared an Arabic context-question-answer dataset
- Cleaned and explored the data
- Used a pretrained transformer question answering pipeline
- Generated predicted answers from Arabic text
- Evaluated predictions using exact match accuracy

## Model Used
- XLM-RoBERTa based question answering model via Hugging Face Transformers

## Tools and Technologies
- Python
- Pandas
- NumPy
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook

## Results
The project demonstrates the ability of transformer models to extract answers from Arabic text and shows a complete NLP workflow from input data to prediction and evaluation.
